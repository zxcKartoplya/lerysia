# Love Postcard on Nuxt 4

Мобильная заготовка под романтичную открытку: фото, письмо, таймер до встречи, блок "скучаю по тебе" и скрытая записка.

## Что менять

Главный файл с контентом: `app/app.config.ts`

Там можно:

- заменить все тексты;
- поменять дату встречи;
- включить или скрыть кнопку с песней;
- подставить свои фото из папки `public/images/`.

## Как подставить свои фото

1. Сложи свои файлы в `public/images/`.
2. Открой `app/app.config.ts`.
3. Замени пути в `hero.imageSrc` и в массиве `gallery`.

Лучше всего смотрятся вертикальные фото с пропорцией `4:5` или `3:4`, потому что сайт сверстан в первую очередь под телефон.

## Команды

```bash
npm install
npm run dev
```

Для статической публикации:

```bash
npm run generate
```

Готовые статические файлы будут в `.output/public`.
